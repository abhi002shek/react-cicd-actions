# Railway Deployment

This metrics server is deployed on Railway for 24/7 monitoring.

## Live Endpoints

- **Health Check**: https://react-metrics-production.up.railway.app/health
- **Metrics**: https://react-metrics-production.up.railway.app/metrics
- **API Info**: https://react-metrics-production.up.railway.app/

## Deploy to Railway

1. Fork this repo
2. Connect to Railway
3. Deploy from `metrics-server` folder
4. Set environment variables if needed

## Local Development

```bash
npm install
npm start
```
