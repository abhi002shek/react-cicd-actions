# Metrics Server for React App Monitoring

This is a simple Express.js server that provides metrics endpoints for Prometheus monitoring.

## Endpoints

- `/health` - Health check
- `/metrics` - Prometheus metrics
- `/api/like` - Record a like (POST)

## Deploy to Railway

1. Go to [Railway.app](https://railway.app)
2. Connect your GitHub account
3. Deploy this folder
4. Set environment variables if needed

## Local Development

```bash
npm install
npm start
```
